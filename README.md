# LivoraPulse (Frontend MVP)
Web-only frontend prototype for **LivoraPulse** using **React + Vite + TypeScript + TailwindCSS + Recharts**.

## Prereqs
Install **Node.js (LTS)**, which includes `npm`.

## Install & run
```bash
npm install
npm run dev
```
Then open the local URL printed by Vite.

## Notes
- Frontend-only MVP, no backend.
- All metrics are dummy JSON stored locally.
- Interactions update local state (no persistence yet).
